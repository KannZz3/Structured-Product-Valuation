# Derivative-Contracts

A collection of concise valuation write-ups on structured derivative products.

## Contents
- **Dual-Directional Buffered PLUS linked to the S&P 500 Index**
- **Walmart Trigger Autocallable Contingent Yield Note**

## Overview
This repository contains PDF reports on selected equity-linked structured products.  
Each report typically covers:
- product structure and payoff
- key contract terms
- pricing assumptions
- valuation methodology
- main valuation results

More derivative contract case studies may be added over time.
